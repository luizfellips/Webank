# Webank
Mini-sistema projetado para estudos.

## Tecnologias
- PHP
- PDO
- MySQL
- HTML
- CSS
- Bootstrap

## Funcionalidades

- Sistema de Login e sessão
- Criar nova conta
- Operações de depósito, transferência e saque
- Registro de transação e tabela de usuários cadastrados
